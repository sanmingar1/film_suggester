# Film Suggester - Búsqueda Semántica de Películas

Aplicación de búsqueda semántica de películas usando RAG (Retrieval Augmented Generation).

## Estructura del Proyecto

```
film_suggester/
├── data/           # Datos de películas (CSV, JSON, etc.)
├── src/            # Código fuente principal
├── utils/          # Funciones auxiliares
├── config/         # Archivos de configuración
├── requirements.txt
└── README.md
```
